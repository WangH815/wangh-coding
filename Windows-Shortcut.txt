Windows-Shortcut


注意事项
a) + 表示组合键,无实际意义;
b) 字母大小写(不能为Shift转的大小写)都可以,为方便识别这里统一大写;
c) 使用编程IDE(集成开发环境),为防止快捷失灵,可将输入法、QQ等软件热键取消;


Ctrl + A          //全部选中当前文件夹下文件和文件夹
Ctrl + C/V        //复制粘贴
Ctrl + X          //剪切
Ctrl + F          //查找;支持文本编辑器、浏览器、IDE和资源管理器等
Ctrl + H          //替换;常用于文本编辑器和IDE中
Ctrl + 鼠标左键   //Ctrl键不松开,鼠标可实现多选;选中目标后鼠标左键拖动(Ctrlb不松开)可建立文件或文件夹副本
Ctrl + N          //打开新窗口;如相同路径资源管理器和新浏览器窗口(不是新标签页)等
Ctrl + Z/Y        //撤销/恢复上一步操作
Ctrl + S          //保存
Ctrl + Tab        //切换应用内窗口;如浏览器等多窗口应用程序;
Ctrl + 鼠标滚轮   //缩放;如浏览器界面、文本编辑器(记事本/Notepad++/UE等)、桌面图标;作用于资源管理器是改变显示方式

Shift + 字母      //字母大小写转换后输入

Alt + Tab         //切换应用(窗口);按下组合键后Alt不松开可使用Tab或左右键切换;区别于Ctrl + Tab
Alt + Enter/双击  //显示选中文件或文件夹(我的电脑/回收站等也可以)的属性
Alt + F4          //关闭应用

Ctrl + Shift + N  //新建文件夹

Ctrl + Alt + S    //文件另存为

Win + A           //打开操作中心
Win + E           //打开资源管理器
Win + R           //打开运行菜单
Win + D           //显示桌面,隐藏所有打开窗口;第二次恢复之前打开的窗口;等价于鼠标点击右下角的键
Win + L           //桌面锁定
Win + I           //打开设置
Win + S           //打开系统快速搜索
Win + K           //打开快速连接无线显示及音频设备
Win + P           //显示扩展(包含无线)
Win + ↑/↓/←/→     //切换当前应用窗口;包含原始窗口、最大化、最小化、左侧和右侧窗口之间的切换

Ctrl + Win + D    //新建虚拟桌面
Ctrl + Win + F4   //关闭当前虚拟桌面
Ctrl + Win + ←/→  //切换虚拟桌面

Win + Alt         //查看所有虚拟桌面

Win + Shift + S   //系统截图工具(win10 支持)

Win + X           //打开隐藏菜单;等价于开始菜单右键
Win + X + T       //打开任务管理器
Win + X + UU      //关机
Win + X + UR      //重启

F2/慢速双击       //选中后重命名文件或文件夹
F11               //浏览器或文本编辑器全屏模式
