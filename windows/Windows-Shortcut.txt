Windows-Shortcuts


注意事项
a) + 表示组合键,无实际意义;
b) 字母大小写(不能为Shift转的大小写)都可以,为方便识别这里统一大写;
c) 使用编程IDE(集成开发环境),为防止快捷失灵,可将输入法、QQ等软件热键取消;

Ctrl/控制键

Ctrl + A          //全部选中当前文件夹下文件和文件夹;A(All)
Ctrl + C/V        //复制粘贴;C(Copy),V(C旁边,使用P不方便)
Ctrl + X          //剪切;X(C旁边)
Ctrl + S          //保存;S(Save)
Ctrl + D          //删除文件文件夹;在文本编辑器或IDE中多为复制粘贴选中行;D(Delete)
Ctrl + F          //查找;支持文本编辑器、浏览器、IDE和资源管理器等;F(Find)
Ctrl + H          //替换;常用于文本编辑器和IDE中;H(R被占用)
Ctrl + N          //打开新窗口;如相同路径资源管理器和新浏览器窗口(不是新标签页)等;N(New)
Ctrl + Z/Y        //撤销/恢复上一步操作
Ctrl + P          //在文本编辑器、浏览器和Office等调用打印功能;P(Print)
Ctrl + Tab        //切换应用内窗口;如浏览器等多窗口应用程序;
Ctrl + 鼠标滚轮   //缩放;如浏览器界面、文本编辑器(记事本/Notepad++/UE等)、桌面图标;作用于资源管理器是改变显示方式
Ctrl + 鼠标左键   //Ctrl键不松开,鼠标可实现多选;选中目标后鼠标左键拖动(Ctrl不松开)可建立文件或文件夹副本


Shift/上档转换键

Shift + 字母      //字母大小写转换后输入


Alt/交替换挡键

Alt + Tab         //切换应用(窗口);按下组合键后Alt不松开可使用Tab或左右键切换;区别于Ctrl + Tab
Alt + Enter/双击  //显示选中文件或文件夹(我的电脑/回收站等也可以)的属性
Alt + F4          //关闭窗口或应用程序

Ctrl + Shift + N  //新建文件夹;N(New)

Ctrl + Alt + S    //文件另存为


Win/窗口键

Win               //打开开始菜单
Win + E           //打开资源管理器;E(Explorer)
Win + R           //打开运行菜单;R(Run)
Win + D           //显示桌面,隐藏所有打开窗口;第二次恢复之前打开的窗口;等价于鼠标点击右下角的键;D(Desktop)
Win + L           //桌面锁定;L(Lock)
Win + I           //打开设置
Win + S           //打开系统快速搜索;S(Search)
Win + ↑/↓/←/→     //切换当前应用窗口;包含原始窗口、最大化、最小化、左侧和右侧窗口之间的切换
Win + K           //打开快速连接无线显示及音频设备;K(Link)
Win + P           //显示扩展(包含无线);P(projection screen,投屏)

Ctrl + Win + D    //新建虚拟桌面;D(Desktop)
Ctrl + Win + F4   //关闭当前虚拟桌面
Ctrl + Win + ←/→  //切换虚拟桌面

Win + Alt         //查看所有虚拟桌面

Win + Shift + S   //系统截图工具(win10 支持)

Win + X           //打开隐藏菜单;等价于开始菜单右键
Win + X + T       //打开任务管理器;T(Task Manager)
Win + X + O       //打开电源选项
Win + X + UU      //关机
Win + X + UR      //重启

F1                //各类软件中的帮助
F2/慢速双击       //选中后重命名文件或文件夹
